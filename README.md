# -E-Commerce-Web-Application
This is a full-stack E-Commerce web application developed using Java (JSP &amp; Servlets) for the backend and MySQL for data persistence. The application simulates a real-world online shopping system with essential features such as user authentication, product browsing, and admin management.
🚀 Features --------
🔐 User Authentication (Login & Signup)
🛍️ Product Browsing with Category Filtering
📦 Order Processing and Confirmation
🧑‍💼 Admin Panel for Product & User Management
💬 Feedback Management System
🔄 CRUD Operations with Database Integration.
🛠️ Tech Stack --------------
Frontend: HTML, CSS, JSP
Backend: Java (Servlets, JSP)
Database: MySQL
Connectivity: JDBC
Architecture: MVC (Model-View-Controller).
📂 Project Structure -------------
/src        → Java Servlets & Backend Logic  
/WebContent → JSP, HTML, CSS Files  
/Database   → SQL Scripts.
